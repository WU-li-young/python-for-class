## 第一支程式
以python所撰寫的hello world 的程式如下:

```
print ("hello world")
```
輸出為
```
hello world
```
相較於其他種程式語言，python為較親人類的程式語言，因此在學習上及程式碼上都比較簡單、精簡。

## 註解
在認真開始之前，先來提最重要的註解。python的註解也有分單行與多行，單行為"#"，多行為" ''' "。
範例:
```
#這是單行註解
```
```
'''
這是多行註解
這是多行註解
這是多行註解
'''
```

## 變數
### 宣告變數
在宣告變數時，電腦會建立一個屬於其的記憶體，各種型態變數有不同的大小。比較特別的式，宣告python的變數時不需要先寫出變數的資料型態，它會自動給予該變數之資料型態。命名時，要注意與其他程式語言相同，不能使用保留字，只能使用ASCII字元!以下是範例:
```
score1 = 99      #int
score2 = 99.9    #float
score3 = "score" #str
score4 = 'a'     #char
score5 = True    #bool
```

### 判斷變數型態:
使用以下的語法來判斷資料的型態，雖然很少用到。
```
score = 1
print(type(score))  
```
輸出為
```
<class 'int'>
```
### 強制轉換資料型態
以下是將`str`改成`int`的範例:
```
num = "50"
print("原本的資料型態是:",type(num))
num = int(num)
print("後來的資料型態是:",type(num))  
```
輸出為:
```
原本的資料型態是: <class 'str'>
後來的資料型態是: <class 'int'>
```

## 輸出輸入
### 輸出
#### 簡單輸出法:
內容與內容間以","隔開。
語法: `print ("字串",變數)`
另外，若要控制其輸出後的格式，可用`end` = `"格式"`，以下為範例:
```
print("Hello World)
```
```
print("Hello","World",end = " ")
```
這兩個的輸出結果式相同的!另外，預設的(若未加上end)print為:`print(end = \n)`
#### 格式化輸出:
|格式 |說明 |
|---|---|
|`%s` |字串  |
|`%d` |整數  |
|`%f` |浮點數|
|`%c` |字元  |

語法: `print("格式"%(資料))`
範例:
```
name = "young"
score = 90
print ("%s gets %d"%(name,score))
```
輸出為:
```
young gets 90
```

##### 控制格式化輸出格式
`%5d`  :向左增加空白直到滿五位
`%-5d` :向右增加空白直到滿五位
`%5.3f`:向左增加空白直到滿五位，但保留3為小數點(四捨五入)
在此說明，前面的數字為你給它的可以輸出空間，小數點後為輸出至的小數點，而正負浩維靠右或靠左對齊。
```
pi = 3.14159
num = 55

print ("%5.3f" % pi)
print ("%5d" %num)
print ("%-5d" %num)
```
輸出為:
```
3.142
  555
555
```

## 條件式
在程式中，我們需要判斷事物，就需要用到`if`的判斷式，語法如下:
```
if 條件:
    流程
```
範例:
```
score = 80
if score >= 60 :
    print("you pass!")
```
輸出為:
```
you pass!
```
還有`elif` 及 `else`，在使用中，`elif`使用於`if`後，其他語法相同。而`else`用於`if`後最後一個，也就是否則怎樣，以下給一個範例比較好理解:
```
score = int(input("請輸入你的成績:"))

if (score >= 80) and (score < 100):
    print("A")
elif (score >= 70) and (score < 80):
    print("B")
elif (score >= 60) and (score < 70):
    print("C")
else:
    print("F")
```
程式會要求你輸入一個成績，並且依成績分數告訴你是等級A或B或C又或是F。
## 陣列

## string字串
string字串就很像一個陣列，以下介紹一些它的用法。
### 取中間其中一個字
`string [第幾位-1]`
注意:[]中第一項皆為0，因此要-1。
範例:
```
string = "Hello World"
print(string[1])
```
輸出為:
```

```
### 取其中幾個字
`string [起始值:終止值:間隔值]`
範例:
```
string = "Hello World"
print(string[0:8:2])
```


## 迴圈
### for
有兩中寫法，簡單來說是這樣:
```
for 條件式:
    內容
```
但進階一點說為以下這樣:
```
for 變數 in range():
    內容
```
或是
```
for 變數 in 陣列:
    內容
```
以下是範例:
```
lst = [0,2,4]
for i in lst:
    print(i)
```
輸出如下:
```
0
2
4
```
### while
而`while`迴圈比`for`迴圈簡單，以下是語法:
```
while 條件式:
    內容
```
範例如下:
```
i = 0
while i < 2:
    print(i)
    i += 1
```
輸出為:
```
0
1
```
### 其他在迴圈中的語法
#### break,continue
`break`和`continue`的功用都是拿來跳出執行中的迴圈的，`break`是跳出整個迴圈，而`continue`則是該次。這樣說可能很難明白，用以下範例來解釋:
```
i = 0
while i < 10:
    i += 1
    if i == 5:
        continue
    print(i,end = ' ')
```
輸出為:
```
0 1 2 3 4
```
由此可見，`break`就是可以直接跳出整個迴圈，在上面的範例中，可見在`i` = `5`時，跳出這個迴圈，也因此沒有繼續印出`5`。
換作是`continue`時，可見以下範例:
```
i = 0
while i < 10:
    i += 1
    if i == 5:
        continue
    print(i,end = ' ')
```
輸出為:
```
1 2 3 4 6 7 8 9 10
```
由此可知，其輸出缺少`5`，是因為`continue`跳過"該次"迴圈所造成的，這樣應該對`break`和`continue`更了解了吧!

## 函式
### 自訂函式
在程式中，若有需要重複執行的事項，移植重寫會很煩又不美觀且難以閱讀。因此使用函式會方便很多。宣告自訂函式語法如下:
```
def 函式名稱 (變數1,變數2,...,變數n):
    執行內容
```
呼叫函式，傳入所需要的值，就像以下:
```
函式名稱 (a,b)
```
需要注意的是，第一次傳入的資料形態要與之後的資料型態相同，也需與函式中所需要的資料型態相同。另外，若不需要傳入資料，()中可以空白。範例如下:
```
def a_times_b(a,b):
    return a*b
a_times_b(5,6)
```
輸出為:
```
30
```
### 引用函式
#### 內建函式庫
python有很多很優良的函式庫，就像math函式庫，以下以引用math函式庫為範例:
```
import math
print(math.pi)
```
要引用的函示各有各的語法，在這裡不多著述。
#### 自己的函式庫
另外，想引用自己所寫的程式也行，方法如下:
```
import 檔案路徑(注1) as 函式名稱

函式名稱()
```
注1:若檔案在相同資料夾下，直接寫出檔案名稱即可，但若為不同資料夾中，就需寫出其路徑!
