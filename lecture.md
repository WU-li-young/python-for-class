## 第一支程式
以python所撰寫的hello world 的程式如下:

```

```
輸出為
```
hello world
```

## 變數
### 宣告變數
在宣告變數時，電腦會建立一個屬於其的記憶體，各種型態變數有不同的大小。另外，python的變數不需要先寫出變數的資料型態，它會自動判斷該變數之資料型態。命名時，要注意與其他程式語言相同，不能使用保留字!以下是範例:
```
score1 = 99      #int
score2 = 99.9    #float
score3 = "score" #str
score 4 = 'a'    #char
score 5 = 0      #bool
```

### 判斷變數型態:
使用以下的語法來判斷資料的型態，雖然很少用到。
```
score = 1
print(type(score))  
```
輸出為
```
<class 'int'>
```
### 轉換資料型態
以下是將`str`改成`int`的範例
```
num = "50"
print("原本的資料型態是:",type(num))
num = int(num)
print("後來的資料型態是:",type(num))  
```
輸出為:
```
原本的資料型態是: <class 'str'>
後來的資料型態是: <class 'int'>
```

## 輸出輸入

### 輸出
#### 簡單輸出法:
語法: `print ("內容",變數)`

#### 格式化輸出:
|格式 |說明 |
|---|---|
|`%s` |字串  |
|`%d` |整數  |
|`%f` |浮點數|
|`%c` |字元  |
語法: `print("格式"%(值列))`
範例:
```
name = "young"
score = 90
print ("%s gets %d"%(name,score))
```
輸出為:
```
young gets 90
```

##### 控制輸出格式
`%5d`  :向左增加空白直到滿五位
`%-5d` :向右增加空白直到滿五位
`%5.3f`:向左增加空白直到滿五位，但保留3為小數點(四捨五入)
```
pi = 3.14159
num = 55

print ("%5.3f" % pi)
print ("%5d" %num)
print ("%-5d" %num)
```
輸出為:
```
3.142
  555
555
```

## if 條件式

語法:
```
if 條件:
    流程
```
範例:
```
score = 80
if score >= 60 :
    print("you pass!")
```
輸出為:
```
you pass!
```
另外，還有elif 及 else，用法相同只是else不能加判斷式。

## string字串
### 取中間其中一個字
`string [第幾位-1]`
注意:[]中第一項皆為0，因此要-1。
範例:
```
string = "Hello World"
print(string[1])
```
輸出為:
```
e
```
### 取其中幾個字
`string [起始值:終止值:間隔值]`
範例:
```
string = "Hello World"
print(string[0:8:2])
```
輸出為:
```
HloW
```

## 迴圈
### for
有兩中寫法，分別為以下:
```
for 變數 in range()
    內容
```
或是
```
for 變數 in 陣列
    內容
```
以下是範例:
```
lst = [0,2,4]
for i in lst:
    print(i)
```
輸出如下:
```
0
2
4
```
